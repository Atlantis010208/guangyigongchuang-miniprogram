# 📦 压缩视频存放目录

将压缩后的视频文件放到这个目录中。

## 📋 文件命名规范

为了方便管理和上传，建议按以下格式命名：

```
课时编号-课时名称_compressed.mp4
```

**示例**：
```
1-1-灯光设计基础_compressed.mp4
2-1-灯光设计软件介绍_compressed.mp4
3-2-PS设计课_compressed.mp4
3-3-灯光设计概念方案_compressed.mp4
3-4-灯光深化设计CAD_compressed.mp4
```

## 📝 使用流程

### 1. 压缩视频

使用 **HandBrake** 或 **FFmpeg** 压缩视频：

**HandBrake（推荐）**：
- 预设：Fast 720p30
- 质量：RF 23

**FFmpeg 命令**：
```bash
ffmpeg -i input.mp4 -c:v h264 -preset medium -crf 23 -c:a aac -b:a 128k output_compressed.mp4
```

详细教程请查看：`VIDEO_COMPRESSION_GUIDE.md`

### 2. 放置文件

将压缩后的视频复制到此目录：

```
20250925-miniprogram-4/
  └─ compressed-videos/
     ├─ 1-1-灯光设计基础_compressed.mp4  ← 放在这里
     ├─ 2-1-软件介绍_compressed.mp4      ← 放在这里
     └─ ...
```

### 3. 运行上传脚本

查看上传配置：
```bash
node scripts/upload-compressed-videos.js
```

### 4. 执行上传

根据脚本输出的命令，使用 MCP 工具上传到云存储。

## ⚠️ 注意事项

1. **保留原始视频备份**：压缩是不可逆的，建议保留原文件
2. **验证压缩质量**：上传前先在本地播放器测试
3. **文件大小检查**：确保压缩后文件大小符合预期（约70%缩减）
4. **不要混合格式**：统一输出为 `.mp4` 格式

## 🎬 快速开始

最简单的方式：

1. **下载 HandBrake**：https://handbrake.fr/downloads.php
2. **打开视频**：选择要压缩的视频文件
3. **选择预设**：右侧选择 "Fast 720p30"
4. **保存路径**：设置输出到此目录
5. **开始压缩**：点击 Start 按钮
