# 问题
六个新增云函数目录缺少 `package.json`，云端无法安装 `wx-server-sdk`，导致运行时报 `Cannot find module 'wx-server-sdk'`。

## 解决方案
- 在以下目录添加 `package.json` 并声明 `dependencies: { "wx-server-sdk": "^2.11.0" }`：
  - `cloudfunctions/orders_create`
  - `cloudfunctions/orders_update`
  - `cloudfunctions/orders_remove`
  - `cloudfunctions/requests_create`
  - `cloudfunctions/requests_update`
  - `cloudfunctions/requests_remove`
- 字段统一：`{ name, version, main: "index.js", dependencies }`。
- 添加后在微信开发者工具逐个执行“上传并部署（云端安装依赖）”。

## 验证
- 通过页面触发云函数（撤销、删除、支付同步）或在“云函数”面板直接触发，日志不再出现缺包错误，返回 `{ success: true, code: 'OK' }`。
