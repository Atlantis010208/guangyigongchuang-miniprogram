## 部署步骤

### 1. 环境准备
1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号（如需发布）
3. 获取小程序的 AppID

### 2. 项目导入
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目根目录：`C:\Users\Administrator\WeChatProjects\miniprogram-4`
4. 输入项目名称和 AppID（测试可使用测试号）

### 3. 图片资源配置
根据 `images/RESOURCES_NEEDED.md` 文件，添加以下必需的图片资源：

**高优先级图片**（影响核心功能）：
- `user-avatar.png` - 用户头像占位符
- `empty-cart.png` - 空购物车状态图
- `search-icon.png` - 搜索图标
- `clear-icon.png` - 清除图标

**中优先级图片**（影响视觉效果）：
- `iphone16pro-hero.png` - 主打产品展示图
- `savings-products.png` - 省钱妙招图片
- 各产品分类图标

### 4. 配置调整
修改 `project.config.json` 中的 `appid` 字段为默认值（可按需替换）：
```json
{
  "appid": "wxe8b6b3aed51577e0"
}
```

同时将云开发环境统一为：`cloud1-5gb9c5u2c58ad6d7`
- 小程序端：在 `app.js` 中将 `wx.cloud.init({ env: 'cloud1-5gb9c5u2c58ad6d7' })`
- 云函数：各 `cloudfunctions/*/index.js` 的 `cloud.init({ env: 'cloud1-5gb9c5u2c58ad6d7' })`

### 5. 功能测试
在开发者工具中测试以下功能：
- [x] 底部标签栏切换
- [x] 产品页面滚动和交互

### 6. 预览和调试

## 技术说明

### 页面路由
- `/pages/products/products` - 产品页面（首页）
- `/pages/explore/explore` - 深入探索
- `/pages/search/search` - 搜索页面
- `/pages/cart/cart` - 购物袋

### 本地存储
项目使用以下本地存储：
- `cartItems` - 购物车商品数据
- `searchHistory` - 搜索历史记录
- `userInfo` - 用户信息
- `hasLaunched` - 首次启动标记

### 样式系统
- 使用苹果设计语言规范
- 支持深色模式自动适配
- 响应式布局适配不同屏幕

## 发布流程

### 体验版发布
1. 在开发者工具中点击"上传"
2. 填写版本号和更新说明
3. 在小程序后台设置为体验版
4. 邀请体验者测试

### 正式版发布
1. 体验版测试无误后
2. 在小程序后台提交审核
3. 通过审核后发布正式版
4. 监控用户反馈和数据
